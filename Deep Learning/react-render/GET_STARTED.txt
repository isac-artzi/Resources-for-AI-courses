┌──────────────────────────────────────────────────────────────────────────────┐
│ 💡 WHAT THE APP DOES                                                         │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  1. User uploads an image (cat, dog, car, etc.)                             │
│  2. React frontend sends image to FastAPI backend                           │
│  3. Backend processes image with PyTorch model (ResNet-18)                  │
│  4. AI identifies what's in the image                                       │
│  5. Returns top 5 predictions with confidence scores                        │
│  6. Frontend displays results beautifully                                   │
│                                                                              │
│  Example: Upload a cat photo →                                              │
│  Results: "tabby cat" (92%), "Egyptian cat" (5%), etc.                      │
